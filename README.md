# AI Imaging Portfolio

Machine learning and computer vision projects for scientific imaging.

This repository contains experiments and projects focused on applying AI techniques to complex scientific imaging datasets.

## Topics

- Ground Penetrating Radar (GPR) imaging
- X-ray tomography
- MicroCT data analysis
- Computer vision for scientific datasets
- Deep learning for image segmentation
- Automated feature detection in imaging data

## Goals

The goal of this repository is to explore how machine learning and deep learning can be applied to scientific imaging problems such as:

- feature detection
- segmentation
- classification
- pattern recognition

## Future Projects

- Segmentation of porosity in microCT data
- Detection of interfaces in GPR images
- Automatic classification of microstructures
- Deep learning for tomography data analysis
